Features

Quick Answers

Select any text and get an instant answer from GPT in a popup.

Context Menu

Right-click selected text and choose Answer with GPT.

Customizable

Choose your GPT model, set max tokens, and manage your API key.

Light & Dark Theme

Switch between beautiful light and dark modes.

Per-site Enable

Enable or disable the extension on specific sites.

Private & Secure

Your API key is stored locally and never shared.

Get Started

  1. Clone the repo: git clone https://github.com/jokelbaf/gpt-quick-reply.git
  2. Open chrome://extensions (or your browser's extensions page).
  3. Enable Developer mode and click Load unpacked.
  4. Select the gpt-quick-reply folder.
  5. Click the extension icon to configure your API key and settings.
